# Phase 0B spike harness

These probes make the Phase 0B architecture decisions reproducible without
changing released scaffolding behavior. They are exploratory evidence, not the
normative registry or projection implementation.

Run the stdlib probes:

```text
python scripts/spikes/phase0b/probe.py
```

Run the Bash manifest probe on Windows Git Bash or Linux Bash:

```text
bash scripts/spikes/phase0b/run_manifest_probe.sh
```
